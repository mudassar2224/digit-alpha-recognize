# 🔥 Digit + Alphabet Recognizer

A Deep Learning web application that recognizes handwritten **digits (0–9)** and **uppercase alphabets (A–Z)** using a CNN model.

## 🚀 Live Demo

https://hhcbfhs4s7n4zxrekx3xub.streamlit.app/

## 📂 GitHub Repository

https://github.com/mudassar2224/digit-alpha-recognize

---

## 📌 Features

* Upload image for prediction
* Camera input support
* Recognizes digits and alphabets
* Top-3 prediction confidence scores
* Smart preprocessing using OpenCV
* Dark mode UI
* Streamlit deployment

---

## 🧠 Model Details

* Deep Learning CNN
* LeNet-5 Inspired Architecture
* TensorFlow / Keras
* Trained on EMNIST Balanced Dataset
* 36 output classes:

  * Digits: 0–9
  * Alphabets: A–Z
